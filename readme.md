## run
Install meteor: 

```bash
curl https://install.meteor.com | /bin/sh
```

Download: 

https://github.com/matth96/Leaflet-with-meteor.git

Go to the project:

```bash
cd path/to/Interactive-map
```

run meteor:
```bash
meteor
```
